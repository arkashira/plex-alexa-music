# Plex Alexa Music

This project enables seamless voice control for music playback in multiple rooms using Alexa devices.

## Features

* Play music in multiple rooms using voice commands
* Specify which rooms to play music in using voice commands

## Requirements

* Python 3.8 or later
* No external dependencies required

## Usage

1. Create a `MusicControl` object with a list of `Room` objects.
2. Use the `play_music` method to play music in specified rooms.
3. Use the `stop_music` method to stop music in specified rooms.

## Testing

Run `python -m pytest` to execute the tests.
